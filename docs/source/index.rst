==================
**IQASucupiraAPI** 
==================

**IQASucupiraAPI** corresponde a uma interface de *chatbot* desenvolvida com o objetivo de facilitar a recuperação de dados de interesse acadêmico através de linguagem natural. Para isso propomos o desenvolvimento de um IQA para acessar os dados presentes na `Plataforma Sucupira`_. Para mais detalhes sobre o uso desta API siga os links disponíveis no sumário abaixo.

..  _Plataforma Sucupira: https://sucupira.capes.gov.br/sucupira/
